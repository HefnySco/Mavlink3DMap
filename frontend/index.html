How to run

Serve this folder with any static web server (ES modules required). Example:

python3 -m http.server 8080
# open http://localhost:8080/frontend/

Keyboard shortcuts

  • F1 Help Toggle
  • 1-9 Goto Drone by Index
  • O next / P previous camera (selected drone)
  • W A S D Q E Change Camera View for Vehicles
  • L Toggle Drone Labels
  • T Toggle Camera Trace
  • R Reset Camera View
  • +/- Change Drones Scale
