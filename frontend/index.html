How to run

Serve this folder with any static web server (ES modules required). Example:

python3 -m http.server 8080
# open http://localhost:8080/frontend/

Keyboard shortcuts

Global

  • 1..9 Focus main camera on drone index
  • O/P Cycle selected attached camera
  • L Toggle labels
  • T Toggle camera helpers
  • -/+ Scale all drones down/up
  • F1 Toggle help dialog

When main camera is selected

  • W/S Move forward/back
  • A/D Strafe left/right
  • H Look at origin
  • Q Reset to default position

When an attached camera is selected

  • W/S Tilt up/down
  • A/D Pan left/right
  • Q/E Roll left/right
  • R Reset orientation
