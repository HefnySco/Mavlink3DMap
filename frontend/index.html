How to run

Using the published package (no install):

npx mavlink3dmap serve -p 8080
npx mavlink3dmap udp2ws --udp-port 16450
# Linux-only streaming
npx mavlink3dmap stream
# All-in-one
npx mavlink3dmap up --port 8080 --udp-port 16450 --stream
# Query Video Device Number
npx mavlink3dmap video-number

Monorepo development shortcuts:

npm run dev         # backend UDP bridge + frontend dev server
npm run devall      # adds camera streaming pipeline (Linux)
npm run start:web   # preview built frontend
